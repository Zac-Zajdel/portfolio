# Capitalize the first letter of a string

## Step 1: Find position of the character we want capitalized

```jsx
const word = 'hello World'
word.charAt(0) // Output: h
```

## Step 2: Capitalize the character we just extracted

```jsx
const word = 'hello World'
word.charAt(0).toUpperCase() // Output: H
```

## Step 3: Grab the remaining string and merge

```jsx
const word = 'hello World'
const solution = word.charAt(0).toUpperCase() + word.slice(1) // Output: Hello World
```
